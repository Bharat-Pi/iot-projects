# Build Any IoT Project using Bharat Pi boards
IoT projects using Bharat Pi for final year students, mini projects, home automation, IoT training and more.

Bharat Pi is an IoT Prototyping platform for students, innovators, startups and enterprises. Bharat Pi boards are available in multiple variants to suit your needs across IoT spectrum of use cases: 
1) 4G/LTE module and ESP32 on a single board enabling rapid prototyping of any IoT use case
2) LoRa module and ESP32 on a single board supporting Smart metering use case
3) Node Wi-Fi with ESP32 for starters

_Launching soon..._ 
1) STM32 Based Node with Wi-Fi (Enterprise Edition)
2) 5G with Wi-Fi and Bluetooth (Enterprise Edition)
3) Edge Data Center (Linux Version)
4) Matter and Zigbee (with Wi-Fi and Storage)
